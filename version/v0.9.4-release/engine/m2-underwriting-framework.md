# M2 承销可行性评估框架

**版本**: v0.9.4-release | **日期**: 2026-07-20 | **定位**: M2 承销视角的债券发行可行性评估框架（发行窗口 · 投资人匹配 · 可比定价）

**所属模块**: 固定收益信用分析引擎 — 多利益方视角（M2 承销商）

---

> **诚实性声明：** 本框架提供承销可行性的**结构化评估方法**，而非发行结果的保证。窗口判断依赖公开可获取的市场数据（一级发行、认购倍数、取消发行占比、中债估值），数据时滞与覆盖缺口按引擎既有"数据缺口即风险信号"纪律处理。本框架不预测市场情绪拐点。

## 一、框架定位与三支柱

承销商（M2）面对的核心问题是：**这只券现在能不能发、发给谁、定什么价。** 本框架把该问题拆为三根支柱：

| 支柱 | 回答的问题 | 主要数据输入 | 输出 |
|---|---|---|---|
| **发行窗口** | 当前市场环境下发行的成功率与成本风险 | 一级供给节奏、认购倍数、取消发行占比、信用利差 | 窗口四档（优/可/慎/避） |
| **投资人匹配** | 哪类机构会买、以什么条件买 | 机构偏好谱、簿记结构、近期可比认购行为 | 目标投资人清单与匹配度 |
| **可比定价** | 合理票面区间在哪里 | 可比券中债估值、期限/评级/行业利差 | 定价区间与偏离校验 |

三支柱**按序消费**：窗口判断决定"要不要发"，投资人匹配决定"发给谁、怎么卖"，可比定价决定"定多少"。任一支柱亮红灯，整体结论降级。

**与 M1 的关系**：M1（债券投资）是二级买方视角的"值不值得买"，M2 是一级卖方视角的"能不能发"——两者互为镜像：M1 的相对价值结论（低估/高估）是 M2 定价区间的验证输入，M2 的窗口判断是 M1 一级参与时机的参考。

## 二、发行窗口判断

### 2.1 三层窗口信号

| 层 | 信号 | 数据形态 | 数据来源 |
|---|---|---|---|
| **市场环境** | 利率周期位置（收益率处于近 1 年分位） | 分位数 | 中债国债/国开收益率曲线 |
| | 信用利差分位（对应评级利差的近 1 年分位） | 分位数 | 中债估值/第三方利差数据 |
| | 一级供给节奏（近 4 周净融资额趋势） | 趋势值 | 交易商协会/交易所发行统计 |
| **政策窗口** | 监管审批节奏（注册/备案通过速度） | 定性+周期天数 | 交易商协会/证监会公示 |
| | 行业准入窗口（行业专项政策鼓励/限制期） | 定性 | 发改委/工信部/行业主管文件 |
| **情绪周期** | 认购倍数（近期可比券平均认购倍数） | 区间值 | 发行公告 |
| | 取消发行占比（近 4 周取消/推迟发行只数占比） | 百分比 | 公开取消公告汇总 |

### 2.2 窗口四档判定

| 档位 | 判定（需同时满足） | 行动建议 |
|---|---|---|
| **优** | 利差分位 ≤30% 且 认购倍数 ≥2x 且 取消占比 ≤5%，无行业准入限制 | 即刻推进，可适当压价 |
| **可** | 利差分位 30-60% 或 情绪中性（认购 1-2x），无重大政策逆风 | 正常推进，定价留安全边际 |
| **慎** | 利差分位 60-80% 或 取消占比 >15% 或 供给拥挤 | 缩短久期/增信后再发，或等待 |
| **避** | 利差分位 >80% 且 情绪冻结（取消占比 >20%），或行业准入收紧期 | 暂缓，改道贷款/非标或等窗口 |

> **判定纪律**：三层信号不一致时，以**情绪周期**为最终裁决层——一级市场真实认购行为是窗口的最直接证据；市场/政策层只作降级依据，不作升级依据。

### 2.3 数据缺口处理

认购倍数与取消占比在部分券种（私募债/PPN）公开覆盖不全——缺口标注后窗口结论降一级置信度（优→可，可→慎），禁止在数据缺口下给出"优"档。

## 三、投资人匹配

### 3.1 机构偏好谱

| 机构类型 | 典型久期偏好 | 评级偏好 | 行业偏好 | 定价敏感度 | 簿记行为特征 |
|---|---|---|---|---|---|
| 商业银行（自营） | 中短（1-3 年） | AA+ 及以上为主 | 城投/金融债偏好 | 中 | 量大、价格引导强、常为主力 |
| 公募基金 | 灵活（1-5 年） | 全谱系（依产品） | 产业债/城投均衡 | 高 | 比价驱动、跟单与撤单都快 |
| 保险资管 | 中长（3-10 年） | AAA/AA+ 为主 | 长周期稳定行业 | 低-中 | 配置型、持有到期、量大稳定 |
| 券商资管/自营 | 中短（1-3 年） | 灵活 | 交易型行业 | 高 | 交易驱动、承销团自联动 |
| 银行理财子 | 中短（1-3 年） | AA+ 及以上 | 城投/央国企 | 中 | 量大、稳健、重票息 |
| 私募/专户 | 灵活 | 可接受低评级 | 高收益产业债 | 低 | 票息驱动、可承接弱资质 |

### 3.2 匹配度评估流程

```
Step 1：从发行人评级/期限/行业 → 初筛偏好谱上的主力机构类型
Step 2：按当前窗口判断修正（窗口差时，仅配置型机构（保险/银行）可依赖）
Step 3：构造目标投资人清单——主力机构（2-3 类）+ 补充机构（1-2 类）
Step 4：评估簿记可行性——基石/锚定可获得性（央国企背景/担保方带动）
```

### 3.3 匹配度等级

| 等级 | 含义 | 承销动作 |
|---|---|---|
| **强匹配** | ≥2 类主力机构有明确偏好且当前窗口可依赖 | 常规簿记，预期足额 |
| **一般匹配** | 仅 1 类主力机构可依赖，或窗口中性 | 需基石/锚定先行，定价留让步 |
| **弱匹配** | 主力机构均不活跃或偏好错位 | 重构条款（缩短期限/增信/分层）或暂缓 |

## 四、可比券定价

### 4.1 可比券选择规则

| 规则 | 要求 | 说明 |
|---|---|---|
| 评级相近 | 同主体评级或 ±1 子级 | 跨 2 档以上不可比 |
| 行业/范式相近 | 同行业优先；无则同范式 | 范式利差大于行业表面利差时需注明 |
| 期限相近 | 剩余期限 ±1 年内 | 期限差异>1 年时按曲线修正 |
| 数量 | 3-5 只 | 少于 3 只时标注"可比不足"，区间加宽 |

### 4.2 定价区间推导

```
基准值 = 可比券中债估值收益率的加权均值（剩余期限加权）
定价区间 = 基准值 ± 调整项合计

调整项：
  流动性溢价：私募/PPN +（估值较公募上浮部分）
  条款溢价：含权/回售/永续特殊条款 ±
  主体差异：发行人与可比券主体的信用差异微调 ±
  情绪溢价/折价：窗口判断联动（优→折价，慎/避→溢价）
```

### 4.3 偏离校验（红旗信号）

| 信号 | 含义 | 处理 |
|---|---|---|
| 拟定价低于可比均值 >30bp | 疑似低价让利/关联认购 | 核查簿记结构，提示定价公允性风险 |
| 一二级倒挂（新发票面 < 二级估值） | 窗口过热或定价偏保守 | 可作卖点，但防"低价发不出高预期"反弹 |
| 可比券估值异常波动（近 2 周 >20bp） | 基准不稳 | 改用区间中值并标注基准可信度 |

## 五、输出规范

**承销可行性结论（三档）**：

| 结论 | 条件 | 含义 |
|---|---|---|
| **可行** | 窗口优/可 + 匹配强 + 定价区间清晰 | 建议按计划发行 |
| **谨慎可行** | 窗口慎 或 匹配一般 或 可比不足 | 附前置条件（增信/缩久期/基石先行/等待窗口） |
| **暂不可行** | 窗口避 或 匹配弱 或 定价基准不可信 | 暂缓，给替代渠道建议（贷款/非标/等待） |

输出物：结论 + 窗口判断依据 + 目标投资人清单 + 定价区间（含推导）+ 风险提示 + 数据完备性与缺口。**模板：Type 16 承销可行性报告。**

## 六、与其他框架的集成

- **M1 相对价值（WP-M1-01）**：互为镜像——M1 的低估/高估结论直接参与 M2 定价区间的情绪调整项；M2 的窗口四档为 M1 一级申购时机提供参照。
- **双轨评级**：发行人主体评级（WP-M0-01 输出）是可比券选择的第一筛选维度。
- **LGFV 特殊考虑**：城投承销需叠加区域利差（同省存量券估值）与区域额度/政策窗口（化债阶段对发行的约束），详见 lgfv-framework.md。
- **集中度约束**：若承销机构自身组合同发行人敞口已高（WP-M4-01 视角），跟投额度需扣减。

## 七、情景示例（匿名）

某 AA+ 级公用事业发行人拟发 3 年期中期票据：

1. **窗口判断**：利差分位 45%（可）、近 4 周可比认购倍数 1.8x（中性）、取消占比 6%（正常）→ 窗口"可"。
2. **投资人匹配**：AA+ 公用事业 3 年 → 银行自营+理财子主力、保险补充；窗口中性 → 匹配"一般"。
3. **可比定价**：选取 3 只同评级公用事业中票（剩余期限 2.5-3.2 年），估值均值 2.85%；主体资质略优于可比均值（-5bp），窗口中性不加价 → 区间 2.75%-2.90%。
4. **结论**：谨慎可行——建议 2.80% 附近开簿，预留 5-10bp 让步空间；若簿记前 1 周认购倍数跌破 1.2x，则降档暂缓。

（本示例为匿名方法演示，不构成任何具体发行建议。）

## 八、质量门

1. **可比券选择合规性**：可比券是否满足评级/行业/期限规则且数量 ≥3；不足时是否已标注"可比不足"。
2. **窗口判断证据充分性**：窗口四档判定是否引用当期市场/政策/情绪数据（非陈旧或主观断言）。
3. **定价区间推导可复核**：基准值与全部调整项是否逐项列明且可追溯。
4. **结论一致性**：结论档位与三支柱结果的映射是否符合 §五 规则（无窗口"避"仍给"可行"等矛盾输出）。

---

## 九、诚实性与局限

- 一级市场情绪可在数日内反转，窗口判断有效期以**周**计，超期需重估。
- 认购倍数/取消占比数据在私募券种覆盖不全，相关结论强制降置信度（§2.3）。
- 本框架不提供发行成功的概率承诺；"可行"结论仍需以簿记当日实况为最终裁决。
